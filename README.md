Olá, bom, esse é foi meu primeiro projeto de 2025; um bot de discord para facilitar minhas mesas de RPG.
Hoje, 12/03/2025; com ajuda do meu professor, consegui terminar as principais coisas do bot:
  - O sistma de ficha: Um comando que mostra a ficha completa do personagem que o usuário possui ou pede
  - Rolagem de dados: Um comando que gira um quantidade de dados que o usuário pedir e soma tudo no final
  - Comunicação estável com o banco de dados: Eu uso um servidor em nuvem (Datastax) para armazenar os dados do bot (skills, personagens...) e consegui manter relação boa entre os comandos do bot e o servidor do banco de dados
